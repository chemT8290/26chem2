# PDF 미리보기 웹페이지 사용 방법

이 폴더의 `index.html`, `26-Q.pdf`, `26-A.pdf` 세 파일을 같은 위치에 업로드하세요.

## GitHub Pages에 올리는 경우
1. 새 저장소를 만들거나 기존 저장소에 세 파일을 업로드합니다.
2. Settings > Pages에서 배포 브랜치를 선택합니다.
3. 발급된 Pages 주소로 접속하면 PDF 미리보기와 다운로드가 작동합니다.

## 주의
- PDF가 보이지 않으면 PDF 파일 권한이 공개되어 있는지 확인하세요.
- `index.html`과 PDF 파일이 서로 다른 도메인에 있으면 브라우저 보안 정책 때문에 미리보기가 막힐 수 있습니다.
- 모바일 브라우저에서는 PDF 미리보기 대신 새 탭 열기 또는 다운로드가 더 안정적일 수 있습니다.
